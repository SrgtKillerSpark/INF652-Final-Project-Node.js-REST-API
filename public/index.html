INF653 — US States REST API

A Node.js + Express + MongoDB API serving data for all 50 US states, with user-contributed fun facts stored in MongoDB.

API Root

All endpoints live under /states/.

GET Endpoints

  • GET /states/ — all state data
  • GET /states/?contig=true — contiguous states only (no AK, HI)
  • GET /states/?contig=false — non-contiguous (AK, HI)
  • GET /states/:state — full data for one state
  • GET /states/:state/funfact — a random fun fact
  • GET /states/:state/capital
  • GET /states/:state/nickname
  • GET /states/:state/population
  • GET /states/:state/admission

POST / PATCH / DELETE

  • POST /states/:state/funfact — body: { "funfacts": ["..."] }
  • PATCH /states/:state/funfact — body: { "index": 1, "funfact": "..." }
  • DELETE /states/:state/funfact — body: { "index": 1 }

Note: the :state URL parameter accepts state codes (KS, NE, TX, NY, etc.) in any case. Indexes on PATCH/DELETE are 1-based.
